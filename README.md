# Vue Simple Task Tracker

## Project setup
```
npm install
npm install -g json server
```

### Compiles and hot-reloads for development
```
npm run serve - to run vue application
npm run backend - to serve json-server
```

### Compiles and minifies for production
```
npm run build
```
